About Us:

The Datalitics project is an attempt to bring visual meaning to select large numbers. Humans are great quickly understanding visual representations.
You can represnent numberical data visually through changes through changes in length, height, color, location, and scale.
We have chosen to use D3, a javascript tool to make visualizations in an attempt to show the depth of some of the datasets we have choosen.
These include mass killing counts, populations, and census data bound to differnet maps.
